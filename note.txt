how to push project in github 
===
git init
git add .
git commit -am "initial commit"
git remote add origin https://github.com/nickldn2211/MovieAppProject.git (the respository link created in github)
git push origin master

run in terminal
====
npx create-next-app@latest
> project <name/>
> typescript No
> eslint Yes 
> tailwind css yes 
>  'src' directory yes 
> app router yes 
> import alias no
==============
cd movie-app
npm run dev
